# RESCAT — App Web con Supabase

Red de apoyo para personas con diabetes tipo 1.

## Cómo correr el proyecto localmente

### Requisitos
- Node.js 18+ → https://nodejs.org

### Pasos

```bash
cd rescat
npm install
npm run dev
```

Abre http://localhost:5173

## Funcionalidades incluidas

- Splash screen animado
- Onboarding de 3 pasos
- Registro real de usuarios (guarda en Supabase)
- Inicio de sesión real con email y contraseña
- Sesión persistente (si cierras y vuelves a abrir, queda logueado)
- Pantalla de inicio con nombre del usuario
- Cerrar sesión

## Publicar en Vercel (gratis)

1. Sube esta carpeta a GitHub
2. Ve a vercel.com → New Project
3. Conecta tu repositorio
4. Haz clic en Deploy

Tu app queda en una URL como: rescat.vercel.app
